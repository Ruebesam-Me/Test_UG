2.3 Navigating, searching and filtering the repository of assessment criteria

2.3.1 Showing or hiding the repository pane

If the repository pane is not visible, click the directory icon underneath the file menu on the left-hand side of the screen to make it visible. You can hide the repository pane using the same icon.

2.3.2 Expanding and collapsing nodes

The repository of assessment criteria is structured by hierarchical nodes. For example, the criterion All SALW shall be marked with a unique identifying mark is located under the following nodes: Marking and record-keeping → Marking → Standards applicable to all markings.

In the repository, you can expand or collapse a main node by clicking the triangular arrow (▸ and ▾) to the left of the checkbox.

Nodes are thematic; for instance, the Marking node contains all criteria in the repository that are applicable to marking.

2.3.3 Searching for assessment criteria

In addition to navigating the tree diagram by expanding and collapsing nodes, you can search for criteria using the search feature:

  1. Click on the magnifying glass icon (🔍) next to the filter icon at the top of the repository pane.
  2. Type the desired keywords or phrase in the search field and press enter.
  3. Criteria containing the search term will be shown in dark typeface with the search term highlighted.
    1. Nodes containing the search term will be expanded in the tree diagram. If a node does not contain the desired keywords or phrase, it will be collapsed.
    2. If the search term is not found, all nodes will remain collapsed.
  4. Click on the "x" when you want to exit the search.
    NB: Searching for a keyword or phrase does not automatically include all relevant criteria in the assessment - you have to select the criteria using the checkboxes.

2.3.4 Filtering assessment criteria

Another method to narrow down and find relevant criteria is to filter the repository by level of recommendation (i.e. shall, should, may, can).

  1. In the left-hand pane, select the filter icon next to the search icon. A dialogue box will appear.
  2. To filter the criteria by level of recommendation, click on the recommendation list and select the desired level of recommendation.
    NB: For an explanation of the levels of recommendation used by MOSAIC, see Clause 3 on Terms and definitions of any MOSAIC module (other than Module 1.20, where they are explained in Clause 2).
  3. Click apply.
  4. To remove the filters, click on the filter icon in the left-hand pane and click on the "Clear" button.
